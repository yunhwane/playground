# 영속성 컨텍스트 

## 비영속
- 객체 생성 후 영속성 컨텍스트에 저장하지 않은 상태 
```java
Member member = new Member();
```

## 영속
- 엔티티 매니저를 통해 컨텍스트에 저장한 상태, 즉 영속성 컨텍스트에 관리
```java
em.persist(member);
```

## 준영속
- 영속성 상태에서 더 이상 관리하지 않은 상태
```java
// 엔티티를 영속성 컨텍스트에서 분리해 준영속 상태
em.detach(member);
// 영속성 콘텍스트를 비워도 관리되던 엔티티는 준영속 상태
em.claer();
// 영속성 콘텍스트를 종료해도 관리되던 엔티티는 준영속 상태
em.close();
```

## 삭제
- 엔티티를 영속성 컨텍스트와 데이터베이스에서 삭제
```java
em.remove(member);
```
